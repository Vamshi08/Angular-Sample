Application Description:

AngularJS Single Page Application that allows a user to Create, List, Update and Delete customers.  A customer is made of up the following attributes:
•	Name
•	Email
•	Telephone
•	Address
	◦	Street
	◦	City
	◦	State
	◦	Zip
The web application uses .json files as the storage mechanism for each customer record.


Installation Instructions:

Pre-requiste: node.js
http://nodejs.org/download/

Unzip the application folder
Launch cmd.exe and navigate to the folder

Run the following commands
npm install
bower install

To start the application run the command
node expressapp.js

Navigate to http://localhost:8001/index.html

All the customer json files are stored in public/customers
